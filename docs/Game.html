Class: Game

Game

Represents all of the functions which generate and control the game board. We use the Tet class to manipulate our Tets.

new Game(canvasId, highScoresListId, devMode)

This function creates a Game class intended to be instantiated by "new Game()". Only one Game per page can work separately at the moment.
Parameters:
Name Type Argument Description
canvasId string This is the id of the canvas element within the document from which this Game class was created.
highScoresListId string This is the id of the list for which we are going to list out the user's past high scores.
devMode boolean <optional>
 This is the option to set the game to be initially in Developer's Mode.
Properties:
Name Type Description
devModeOn boolean If this is true, we can do Developer's Only events.
BOARD_ROW_NUM number Pseudo-constant set to 16, determined by Tetris standards set in place before my game was made.
BOARD_COL_NUM number Pseudo-constant set to 10, determined by Tetris standards set in place before my game was made.
newTet boolean If true, we want to create a new Tet.
currentTet Tet This is the Tet that is falling and controlled by the user.
nextTet Tet This is the Tet that is going to come out after our currentTet lands.
updateLanded boolean If true, we should update our landed array to be used in collision detection.
allTets Array.<Tet> This is the array of all Tets that are in the game.
tetsToRemove Array.<Tet> This is the array of all Tets that need to be removed before being drawn.
score number This is the score that we're going to use to display.
updateScore boolean This is the boolean we check to see if we should update our high score list or not.
dropInterval number This is the interval, in milliseconds, for which our currentTet is going to drop 1 block.
gameOver boolean If this is set to true, we handle the "game over" events.
canvasWidth number This is the width that we set. This width can be adjusted and our game will scale to it.
blockS number This is the length of the side of each "block" on the game, in pixels.
canvas Element This is the DOM element for which we are going to be drawing on.
panelHeight number This is the height of the panel which houses our score, nextTet, and PAUSED/DEV text.
landed Array.<Array.<number>> This is the array of array of numbers which we are going to populate with our allTets to be able to detect Tet collision.
highScoresListId string This is the name of the high score list DOM element for which we are going to show our user their past high scores.

handleEvents()

This method creates 3 event listeners. 2 for the window and 1 for the document. The 2 events for the window are onblur and onfocus. These will pause the game when you leave the game window and resume it when you come back. The event for the document listens for onkeydown events. These basically allow the user to interact with the game.
